# AGENTS.md — Codex App adapter for Taste Skill frontend image workflow

## Purpose
Use the imported frontend image skill as a **premium frontend reference discipline**, not as a generic image-generation gimmick.

The core value to preserve from the source material is:
- image-first website concept generation
- premium, art-directed frontend taste
- anti-slop visual discipline
- implementation-friendly output
- strong spacing, hierarchy, rhythm, and composition

## Operating rules
1. Think before acting.
2. Start with the minimum viable implementation.
3. Only modify what is necessary.
4. Define verification before declaring completion.
5. If uncertainty materially affects correctness, ask.
6. Do not perform unnecessary refactors.

## How to apply this skill
When the task is frontend-heavy:
- prefer premium, implementation-friendly design references over generic UI filler
- avoid repetitive AI aesthetics, dashboard-card spam, weak hierarchy, and dense clutter
- preserve strong whitespace, visual rhythm, and image-led composition
- keep outputs structured enough that a developer can implement them

## Preferred workflow
1. Clarify the page or section goal.
2. Decide whether image-first ideation is actually useful.
3. If yes, derive a concrete design direction:
   - theme paradigm
   - hero structure
   - section rhythm
   - typography character
   - image treatment
4. Convert the direction into minimal implementation steps.
5. Verify layout, hierarchy, spacing, and readability after changes.

## Anti-slop constraints
- No generic purple/blue AI glow unless explicitly justified.
- No cloned section patterns without reason.
- No dense, overfilled layouts.
- No weak headline hierarchy.
- No image use as meaningless decoration.
- No implementation-unfriendly mood art when the task is frontend.

## Verification checklist
Before completion, confirm:
- hierarchy is obvious
- hero is clean and intentional
- imagery supports structure
- section spacing is controlled
- layout is implementable
- output avoids obvious AI-template patterns
