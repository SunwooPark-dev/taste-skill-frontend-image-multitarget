# Quick Start

## Fastest path
1. Open `docs/migration/FINAL_PROMPTS.md`
2. Choose your target:
   - Codex App
   - Antigravity App
   - Gemini terminal workflow
3. Copy the matching prompt block
4. Run one scenario from `docs/migration/SCENARIOS.md`
5. Record results in the corresponding `*_RESULTS.md`

## Best first target
Start with Codex App if you want the fastest feedback loop.
